Welcome to our Cloud Computing PBL project! This platform demonstrates real-time multiplayer gaming using modern web technologies and cloud infrastructure.

• Backend: Node.js with Express

• Real-time Communication: Socket.io

• Deployment: Cloud platform

Project Features

• Real-time multiplayer Tic Tac Toe

• Automatic room creation and sharing

• Responsive design for all devices

• Clean, modern UI with monospace typography
